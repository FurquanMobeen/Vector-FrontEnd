# CLIP Vector Search - Next.js Frontend

> **🎉 Successfully migrated from Streamlit to Next.js with TypeScript!**

A modern, full-stack CLIP-based image search application with user authentication and vector search capabilities. The frontend is now built with Next.js 15, TypeScript, and shadcn/ui.

## Quick Start

### Prerequisites
- Node.js 18+ and npm
- FastAPI backend running at `http://localhost:8000`

### Installation

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   ```

3. **Open application**:
   Navigate to [http://localhost:3000](http://localhost:3000)

4. **Login**:
   - Username: `admin`
   - Password: `admin123`

## Features

✅ **Image Search** - Upload an image to find visually similar images
✅ **Text Search** - Search for images using natural language
✅ **Image Management** - Upload, edit metadata, and delete images
✅ **Authentication** - Secure login with JWT tokens
✅ **Modern UI** - Dark theme with purple/pink gradient accents
✅ **Responsive** - Works on desktop, tablet, and mobile

## Tech Stack

**Frontend**: Next.js 15, TypeScript, shadcn/ui, Tailwind CSS
**Backend** (Required): FastAPI, CLIP, ChromaDB, JWT auth
